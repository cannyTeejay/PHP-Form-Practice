# PHP Form Handling Practice

A beginner PHP project demonstrating:
- HTML forms
- POST request handling
- Input sanitization
- Basic PHP backend logic

## Technologies
- PHP
- HTML
- CSS
- XAMPP (Apache)

## How to Run
1. Install XAMPP
2. Clone the repository into `htdocs`
3. Start Apache
4. Open `http://localhost/php-form-practice`
